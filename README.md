# CodeStart-website
## How to Run the Website
1. Open a terminal and navigate to the backend project directory:

Example (Windows):

cd path\to\CodeStart-website\backend

2. Install backend dependencies:

npm install

3. Start the backend server:

node index.js

4. Now, open a new terminal and navigate to the frontend directory:

cd path\to\CodeStart-website\frontend

5. Install frontend dependencies:

npm install

6. Open your browser and go to:

http://localhost:4200/
